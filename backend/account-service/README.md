# Account Service

## API

- `GET /accounts/:id`
  - Returns an account with the given id.
  - `id`: Account id.
- `POST /accounts`
  - Creates a new account.
- `PUT /accounts/:id`
  - Updates an account with the given id.
  - `id`: Account id
- `DELETE /accounts/:id`
  - Deletes an account with the given id.
  - `id`: Account id
